About

Muhammad Zaini is a game developer with a primary focus on Unity, specializing in gameplay mechanics implementation and UI systems. He has a strong understanding of game logic, system driven design, and translating game concepts into reliable in game mechanics.

He is experienced with C# and supporting languages such as Python and JavaScript, and is comfortable working within structured game systems, including turn based, grid based, and RPG mechanics. In addition to mechanics, he has a solid foundation in UI and UX principles to ensure clarity and smooth player interaction.

Zaini also has experience in storytelling, which supports his work on narrative driven games, allowing mechanics, UI, and narrative to work cohesively.
